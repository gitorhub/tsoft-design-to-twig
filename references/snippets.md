# Snippets Uygulama Rehberi (`snippets/**/*`)

## Yol Esleme
- `snippets/product-list/product-card/*.twig` -> `references/snippets/product-card.md`
- `snippets/header/mobile-menu/*.twig` -> `references/section/header.md`
- `snippets/product-detail/**/*` -> `references/section/product-detail.md`
- `snippets/showcase/*.twig` -> `references/section/showcase.md`

## Genel Kurallar
- Mevcut snippet adlarini ve include yollarini degistirme.
- `SETTING.SUBFOLDER_*` fallback kullanilan yerlerde mantigi koru.
